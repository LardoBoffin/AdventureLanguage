to create a project

cd to application distribution folder, e.g. 

cd /Users/philjones/Documents/z 


dotnet al.dll -new /users/philjones/documents Ransom


to build a project

cd to application distribution folder, e.g. 

cd /Users/philjones/Documents/z 

dotnet al.dll -b /Users/philjones/Documents/Opus2
